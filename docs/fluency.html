f l u e n c y

An HTML, CSS, and Javascript
framework for interactive video

Visual languages exist for film and design.
Fluency is a new form of communication.

Introduction

Fluency provides independent filmmakers with everything they need to make an interactive film. Anyone that knows basic HTML and CSS can use Fluency to make an interactive film.

The name refers to the visual language of film. User interface design has its own visual language. Fluency is designed to promote and standardize a new language for interactive film.

Getting started

It's easy to install Fluency. Download and unzip the the framework. Remove the Javascript and CSS files, place them in the appropriate directories, and link to them in the header of your page.

Example Copy 
<link rel="stylesheet" type="text/css" href="css/fluency.css"/>
<script type="text/javascript" src="js/fluency.js"></script>

Contribute

Hackathons

Fluency is an open source framework available for use under the MIT license. Here's a link to the project on GitHub. We also hold hackathons. If you live in the San Francisco Bay Area and want to join us next time, join our meetup group.

Mobile is not first here

While it's important to design web pages and applications for mobile use, interactive film is different. Future versions of this framework will embrace mobile, especially when it comes to graceful degradation, but we need to get the core framework working before designing for other platforms.

Submitting pull-requests

The most effective way to contribute is to attend our hackathons, but feel free to submit pull-requests for us to review. Reach out to us if you want to make a major contribution or influence the direction of the framework.

Small and lighten

Often used in conjunction with the .lighten class, the <small> tag reduces the size of the of text by 85%. This is a great way to include relevant details underneath a heading.

Example Copy

Nullam dictum felis eu pede mollis pretium

<p class="small lighten">Nullam dictum felis eu pede mollis pretium</p>

Links

Standard <a> tags are included in blue with a text-decoration: underline; hover state. Icon fonts are paired in proportion to links when nested inside of <a> and <span> tags.

Example Copy
Standard link
Icon link pair

<span class="icon-pair"><a><i class="fa fa-play-circle"></i> Icon link pair</a></span>

Branded loader

Video requires bandwidth. This makes a loader a large part of the user experience. Swap out our logo for your own by editing the <img> tag.

Example Copy

</div>
